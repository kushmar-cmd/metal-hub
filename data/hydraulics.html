נוסחאות הליבה
// עיקרון פסקל — לחץ בנוזל סגור מועבר שווה לכל כיוון
F = P × A   // כוח [N]. P=[Pa], A=[m²]
P = F / A   // לחץ [Pa=N/m²]
// המרות לחץ נוחות
1 bar = 100,000 Pa = 10.2 kgf/cm² = 14.5 PSI
1 MPa = 10 bar = 145 PSI
// ספיקה ומהירות
Q = A × v   // [m³/s] → ×60000 = [L/min]
v = Q / A   // מהירות בוכנה מהספיקה
// הספק הידראולי
P_kW = p[bar] × Q[L/min] / 600
// שטחי בוכנה (צילינדר עם מוט)
A_ext = π×D²/4   // שטח הרחבה (מלא)
A_ret = π(D²−d²)/4   // שטח כיווץ (מינוס מוט)
▸ שטח הכיווץ תמיד קטן מההרחבה → מהירות כיווץ מהירה יותר באותה ספיקה, אך כוח נמוך יותר!
▸ יחס שטחים טיפוסי (Regeneration): 1.6:1 עד 2:1
