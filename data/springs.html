⬇️ קפיץ לחיצה — Compression Spring Design
פרמטרים גיאומטריים
ISO 10243
d — קוטר חוט0.1-50mm (לפי עומס)
D — קוטר ממוצעD = D_outer - d = D_inner + d
C = D/dאינדקס קפיץ: 4-12 אידיאלי
n — כריכות פעילות3-20 טיפוסי
n_totaln + 2 (Closed Ground קצה)
Ls = d × n_totalאורך מוצק Solid Length
Lf = Ls + δ_max + clearanceאורך חופשי
נוסחאות מאמץ
Wahl Correction
k = G×d⁴/(8×D³×n)קצב קפיץ [N/mm]
Kw = (4C-1)/(4C-4) + 0.615/Cגורם Wahl
τ = Kw × 8FD/(πd³)מאמץ גזירה [MPa]
δ = F/k = 8FD³n/(Gd⁴)סטייה [mm]
τ_static ≤ 45% Sutגבול מאמץ סטטי
τ_fatigue ≤ 35% Sutגבול מאמץ עייפות
Buckling — כיפוף לרוחב
L/D > 4 → סכנה!
תנאי BucklingL₀/D > 2.62 (Free-Free) · >4.0 (Free-Fixed)
פתרונותמדריך פנימי (Arbor) · מדריך חיצוני · Conical
Arbor קוטרDi - 1.0 עד Di - 0.5 mm
Guide I.D.Do + 1.0 עד Do + 0.5 mm
כלל מעשי: L/D ≤ 3 בטוח · 3-4 בדוק Buckling · >4 = השתמש במדריך

Rules of Thumb
אינדקס C4 ≤ C ≤ 12 (אידיאלי C=6-9)
רוחב כריכהפיסה ≥ 0.1×d בין כריכות בדחיסה מלאה

Natural Frequencyfn = (d/πDn²) × √(G/8ρ) → לא תהדהד!
ייצוב ב-60°C+Stress Relax → בחר חומר מתאים

🔄 קפיץ פיתול — Torsion Spring Design
נוסחאות קפיץ פיתול
כיפוף לא גזירה!
k_θ = E×d⁴/(10.8×D×n)קצב [N·mm/°]
T = k_θ × θמומנט [N·mm], θ=[°]
Ki = (4C²-C-1)/(4C(C-1))גורם ריכוז פיתול
σ = Ki × 32T/(πd³)מאמץ כיפוף [MPa]
σ ≤ 0.78×Sutגבול מאמץ סטטי
E=207GPa לפלדה (לא G!) — כי מאמץ כיפוף
שינויי D תחת עומס
חשוב לעיצוב!
RH Winding (סובב CW)D מצטמצם בטעינה
LH Winding (סובב CCW)D מתרחב בטעינה
D_loadedD' = D×n / (n + θ/360°)
Arbor קוטרDi_loaded - 0.5mm לפחות
⚠️ Arbor חייב לאפשר הידוק D בטעינה!
