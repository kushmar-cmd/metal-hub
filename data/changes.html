  · גורם מגיש: הנדסה / ייצור / לקוח
  · תיאור שינוי + נימוק + השפעות
  · מספר חלק + Rev נוכחי
2. ניתוח השפעה
  · חלקים מושפעים (where-used)
  · מלאי קיים — מה לעשות עמו?
  · עלות + לוחות זמנים
3. אישור (ECO Board)
  · הנדסה + ייצור + רכש + אחסון
  · תאריך כניסה לתוקף
4. יישום
  · עדכון שרטוט + Rev
  · עדכון BOM / דרישות
  · הודעה לכל המחלקות
5. סגירה ותיוק
  · רשומה ב-PLM/ERP
  · שמירת Baseline
